# z-int-proj

# to run server

## cd into server

cd /server

## install dependencies

yarn add

## add .env file

in the src folder (in server repo)
add a .env
insert the following
API_KEY=<'your ticketmaster-api-key-goes-here>
save file

## run

yarn run
